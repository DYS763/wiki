# Header

- 这个游戏的小数点是逗号`,`，千位分隔符是点`.` ~不要问为什么~
- 殖民要带殖民船 ~什么时候改进错误消息~
- 概览页面（最左上角的按钮）点击行星的名字可以改名 ~也可以删除，请不要仔细思考删除是什么操作~
- 控制页面（概览页面按钮下面）和资源页面（建筑按钮右边那个图标）可以看到一些统计信息 ~这些巨大的表格甚至不能告诉你这个格子是哪一行那一列的~
